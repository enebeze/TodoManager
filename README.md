# TodoManager

Project TodoManager of React Native 1 - Pos Iesb

## Install

- Installing dependencies:

```{r, engine='bash', count_lines}
$ yarn
```

- IOS

```{r, engine='bash', count_lines}
$ cd ios && pod install && cd ..
```

## Running


```{r, engine='bash', count_lines}
$ yarn start
$ yarn ios or yarn android
```
